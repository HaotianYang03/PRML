## 🚀Getting Start

### 1. Clone the repository

```python
git clone https://github.com/HaotianYang03/PRML
cd PRML
```

### 2. Create environment

```python
conda env create -f environment.yml
```

### 3. Run the baseline

```python
python feature0526.py
python test06052.py
python trainOUT.py
```
